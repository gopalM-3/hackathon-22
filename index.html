About The Event

Hacktoberfest, the month-long festival of supporting and contributing to open-source, is back again this year! We are excited to share with you guys that CBIT will be hosting a 24-hour online hackathon as part of it, called CBIT Hacktoberfest Hackathon. Hacktoberfest is a virtual festival to celebrate open source contributions presented by DigitalOcean, appwrite and docker. It is the easiest way to get into open source!

Where

✓ It is a virtual event hosted on Github and Discord

When

✓ We are hosting the Hacktoberfest on the 29th and 30th of October 2022.
